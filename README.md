# Intelligent-customer-support-ticket-router-
Hybrid ML &amp; MCP Ticket Routing System automates customer support ticket classification using NLP, TF-IDF vectorization, and similarity-based machine learning. Low-confidence predictions are routed through an MCP-powered LLM fallback for intelligent analysis. The system improves routing accuracy, reduces manual effort, and optimizes AI usage costs.
